# 名片辨識與通訊錄管理

## 任務目標

辨識使用者上傳的名片照片，萃取聯絡資訊，並安全地寫入通訊錄 Excel 檔案，避免重複新增。

---

## 固定規格

### 輸出檔案
- **檔案名稱**：`公司通訊錄.xlsx`
- **位置**：使用者選擇的工作資料夾（`/sessions/happy-blissful-clarke/mnt/115Q2-3/`）

### 欄位定義（固定順序）

| 欄位 | 說明 |
|------|------|
| 姓名 | 名片上的中文或英文姓名 |
| 公司 | 公司或機構名稱 |
| 職稱 | 職位或頭銜 |
| 手機 | 行動電話號碼 |
| 市話 | 市內電話（含分機） |
| Email | 電子郵件地址 |
| 地址 | 公司或通訊地址 |
| 建立日期 | 自動填入今天的日期（格式：YYYY-MM-DD） |

- 辨識不到的欄位一律填入 `—`（全形破折號），不可留空

---

## 執行步驟

### Step 1：辨識名片圖片

仔細觀察使用者上傳的每一張名片圖片，逐一萃取以下資訊：

- 掃描圖片中的**所有文字區塊**
- 根據版面位置與字體大小判斷各欄位歸屬
- 若名片為英文，將姓名欄位保留原文；若為中文名片，保留中文
- 電話號碼需區分手機（09 開頭或含 +886-9）與市話（02、03… 開頭）
- 若一張名片有多個電話，依類型分配；若同類型超過一組，以逗號分隔填入同一欄
- 地址盡可能完整擷取，包含郵遞區號

### Step 2：讀取現有通訊錄（去重檢查）

使用下方的 Python 腳本讀取現有的 `公司通訊錄.xlsx`（若不存在則自動建立新檔）：

```bash
python3 /sessions/happy-blissful-clarke/mnt/.claude/skills/business-card/scripts/upsert_contacts.py \
  --file "/sessions/happy-blissful-clarke/mnt/115Q2-3/公司通訊錄.xlsx" \
  --contacts '[{"姓名":"王小明","公司":"測試公司","職稱":"工程師","手機":"0912345678","市話":"02-12345678","Email":"test@example.com","地址":"台北市","建立日期":"2026-04-22"}]'
```

- 腳本會自動比對「姓名 + 公司」是否已存在
- 重複的聯絡人不新增，並回傳跳過的名單
- 新聯絡人接在最後一列之後

### Step 3：回報結果

完成後，向使用者報告：

1. ✅ **成功新增**的聯絡人姓名與公司
2. ⚠️ **已跳過**（重複）的聯絡人，並說明原因
3. 提供 Excel 檔案的下載連結（使用 `computer://` 格式）

---

## 注意事項

- **不要覆蓋**現有資料，永遠只在最後一列新增
- 若使用者一次上傳多張名片，全部辨識完畢後**批次寫入**，一次執行腳本
- 日期格式固定為 `YYYY-MM-DD`（例如：`2026-04-22`）
- 若圖片模糊或無法辨識，告訴使用者哪張名片辨識失敗，並請他確認
- 回覆使用**繁體中文**，語氣友善專業
