# 115Q2-3: Claude Cowork 實戰工作坊 — 製造業主管篇

## 課程簡介
本課程專為製造業（豆腐機械設備）的中高階主管設計，涵蓋業務、研發、行銷、電機四大部門。
透過 8 個實戰任務與 12 個 Skill SOP 範本，讓主管們在 2 小時內學會將日常重複文書工作「外包」給 AI 代理人。

## 適用對象
- 業務部主管：報價、出貨、客戶管理
- 研發部主管：技術規格書、BOM 表管理
- 行銷部主管：銷售報表分析、客訴處理
- 電機部主管：品管檢查表、設備維護紀錄

## 課程時長
約 120 分鐘（含動手操作與 10 分鐘休息）

## 課程架構
| 階段 | 內容 | 時間 |
|------|------|------|
| 第一階段 | 觀念破冰（製造業痛點開場） | ~15 分 |
| 第二階段 | 環境設定 + 安全工作區 | ~10 分 |
| 第三階段 | 基礎實戰 — 任務 1~4  | ~40 分 |
| ☕ 休息 | — | ~10 分 |
| 第三階段 | 產業實戰 — 任務 5~8  | ~30 分 |
| 技能工具庫 | Skill 安裝教學 + 12 個 SOP 展示 | ~10 分 |
| 第四階段 | 進階展示 + 結訓挑戰 | ~5 分 |

## 實戰任務列表
1. 📂 檔案收納術（全部門）
2. 📝 品管檢查表轉電子檔（電機/研發）
3. 🧾 客戶 PO 直出出貨單（業務）
4. 📅 產線異常會議待辦追蹤（全部門）
5. 💰 歷史報價 → 新報價單（業務）
6. 🔧 中文規格書 → 英文技術文件（研發/電機）
7. 📊 Q1 銷售報表 → 主管摘要（業務/行銷）
8. 📧 客訴信 → 回覆信 + 追蹤表（業務/行銷）

## Skill SOP 範本（12 個）

### 通用辦公（8 個）
| 觸發詞 | 功能 |
|--------|------|
| `/名片` | 名片照片 → 公司通訊錄 Excel |
| `/追蹤信` | 會議紀錄 → 各負責人提醒 Email |
| `/主管摘要` | 多份報表 → 一頁 A4 主管摘要 |
| `/報帳` | 發票收據 → 年度報帳總表 |
| `/白板` | 白板照片 → 專案進度表 Excel |
| `/交接` | 工作資料夾 → 完整交接文件 |
| `/報價` | 歷史報價 → 新報價 Excel |
| `/LINE總結` | LINE 匯出 txt → 討論總結與分工 |

### 製造業專屬（4 個）
| 觸發詞 | 功能 |
|--------|------|
| `/出貨` | 客戶 PO → 公司出貨單 |
| `/規格翻譯` | 中文規格書 → 英文 Technical Spec |
| `/客訴` | 客訴內容 → 回覆信 + 內部追蹤表 |
| `/BOM比對` | 新舊 BOM 表 → 差異分析報告 |
