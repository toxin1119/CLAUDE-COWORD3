# /報價
> 🏷️ **Skills UI 英文名稱（方法一手動建立時填入）**：`quotation`
> 📌 **觸發詞（方法二動態載入後使用）**：輸入 `/報價`

## 指令
參考過去的報價單範例，以及這次新專案的溝通紀錄，幫我整理出一份新的 Excel 報價單。

## 執行規則
- 固定寫入檔案：參考之前的報價格式，產生並命名為 `報價單_專案名稱_YYYY-MM-DD.xlsx`
- 維持原本報價單的外觀排版、字體及表格線條
- 將新專案所需的品項與對應金額填入
- 如果對話中沒有提到具體金額的品項，請照舊報價依比例估算，並在備註註明「[AI預估數字需確認]」
- 確認最後的 SUM 公式正確計算總額
