# /LINE總結
> 🏷️ **Skills UI 英文名稱（方法一手動建立時填入）**：`line-summary`
> 📌 **觸發詞（方法二動態載入後使用）**：輸入 `/LINE總結`

## 指令
這是一份剛從 LINE 群組匯出的對話紀錄（.txt），幫我總結一下會議重點與後續待辦。

## 執行規則
- 產出一份 Word 格式的「討論總結與待辦清單」
- 第一部分：一句話總結這個群組在討論的核心重點
- 第二部分：過濾掉貼圖、打招呼、無關緊要的閒聊；若出現大量「哈哈」、「好喔」、「收到」等社交回應，一律略過不列入
- 第三部分：重點列出**「誰被分配到了什麼任務」**（任務名稱、負責人、期限）
- 若有人被提及但沒壓期限，預設標註「請確認期限」
- 語氣：專業、條理分明
- 檔案命名格式：`LINE討論總結_YYYY-MM-DD.docx`
