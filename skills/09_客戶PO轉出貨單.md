# /出貨
> 🏷️ **Skills UI 英文名稱（方法一手動建立時填入）**：`shipment`
> 📌 **觸發詞（方法二動態載入後使用）**：輸入 `/出貨`

## 指令
讀取客戶的採購單（PO），幫我把資訊填入公司標準的出貨單模板中。

## 執行規則
- 讀取 `data` 資料夾中的 `出貨單.xlsx` 空白模板
- 從客戶 PO 中擷取：客戶名稱、聯絡人、送貨地址、訂單編號、品項明細（品名、型號、數量、單價）
- 將資訊填入出貨單的對應欄位，**嚴禁破壞原始格式與框線**
- 自動計算小計、稅額（5%）、總計
- 出貨日期預設為今天，交貨條件填「工廠交貨 (EXW)」
- 若 PO 上有特殊備註（如指定包裝方式、分批出貨），務必填入備註欄
- 若金額或數量無法辨識，在備註欄標注「⚠️ 需人工確認」
- 檔名格式：`出貨單_客戶名稱_YYYY-MM-DD.xlsx`
- 原始空白模板不可被覆蓋
