# /規格翻譯
> 🏷️ **Skills UI 英文名稱（方法一手動建立時填入）**：`spec-translate`
> 📌 **觸發詞（方法二動態載入後使用）**：輸入 `/規格翻譯`

## 指令
讀取這份中文的設備技術規格書，幫我翻譯成專業的英文 Technical Specification 文件。

## 執行規則
- 產出格式：Word (.docx)，含封面頁
- 封面頁包含：產品型號、產品名稱（中英對照）、版本號、日期
- 專業術語翻譯規則：
  - 磨漿機 → Soy Milk Grinding Machine / Soy Grinder
  - 豆腐成型機 → Tofu Forming Machine / Tofu Press
  - 壓榨機 → Hydraulic Press Machine
  - 濾漿機 → Soy Milk Filter / Pulp Separator
  - 煮漿機 → Soy Milk Cooker / Boiling Tank
  - 凝固劑 → Coagulant
  - 不鏽鋼 → Stainless Steel (SUS304/SUS316)
  - 馬力 → HP (Horsepower)
  - 產能 → Capacity (kg/hr 或 L/hr)
- 數值單位保留公制，但括號內附上英制對照（如適用）
- 技術參數表格保留原始排版結構
- 若原文有模糊之處，翻譯時在旁加註 `[Note: original text unclear]`
- 檔名格式：`Technical_Spec_產品型號_YYYY-MM-DD.docx`
