# /BOM比對
> 🏷️ **Skills UI 英文名稱（方法一手動建立時填入）**：`bom-compare`
> 📌 **觸發詞（方法二動態載入後使用）**：輸入 `/BOM比對`

## 指令
比對這兩份 BOM 表（物料清單），找出所有差異並產出一份差異分析報告。

## 執行規則
- 讀取資料夾中的兩份 BOM Excel 檔（依檔名或版本號判斷新舊）
- 比對項目：料號、品名、規格、數量、單位、供應商
- 差異分類：
  - 🟢 **新增料件**：新版有但舊版沒有的
  - 🔴 **刪除料件**：舊版有但新版沒有的
  - 🟡 **變更料件**：同一料號但規格/數量/供應商有異動
- 產出格式：Excel (.xlsx)
  - Sheet1「差異總覽」：用顏色標記三種差異類型
  - Sheet2「完整新版 BOM」：含所有料件
  - Sheet3「影響評估」：條列每項變更可能影響的面向（成本、交期、品質）
- 在 Sheet1 最上方加一行摘要：「共 N 項變更：新增 X 項、刪除 Y 項、變更 Z 項」
- 若料號格式不一致導致無法比對，標註「⚠️ 料號格式不符，請人工確認」
- 檔名格式：`BOM差異分析_產品型號_YYYY-MM-DD.xlsx`
