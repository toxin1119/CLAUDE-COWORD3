## 二、現有頁面問題

1. **缺乏「為什麼需要 Skill」的衝擊感** — 直接跳到三步驟，學員沒有動力
2. **工具卡片只有名稱和說明** — 看不出「輸入什麼、輸出什麼」
3. **安裝步驟是純文字列表** — 視覺層次不夠清晰

## 三、改版內容（共三處）

### 改動一：在開頭加「有 / 無 Skill 對比」卡片

**插入位置：** 緊接在 `<div class="lesson-content">` 之後，現有內容之前

**視覺設計：** 紅 / 綠兩色卡片並排（各佔 50%）

```
┌─────────────────────────────┐  ┌─────────────────────────────┐
│ 😩 沒有 Skill 的每一次       │  │ 🚀 裝了 Skill 之後           │
│ 「請幫我把客戶 PO 轉成出貨   │  │                             │
│ 單，格式是 A 公司的模板，    │  │ 打 /出貨                     │
│ 欄位要有品名、數量、單價、   │  │ → 拖入客戶 PO               │
│ 稅額，稅率 5%……」            │  │ → Enter                     │
│                             │  │ → 自動完成 ✅               │
│ → 每次都要重新解釋           │  │ → 教一次，永遠記得           │
└─────────────────────────────┘  └─────────────────────────────┘
```

**實作方式：** `display:grid; grid-template-columns:1fr 1fr`，左卡背景 `#fff1f2`（紅），右卡背景 `#f0fdf4`（綠）

---

### 改動二：每張工具卡新增「輸入→輸出」流程條

**插入位置：** 每張 `<a>` 工具卡的 `<p class="text-gray-500 text-sm">` 描述文字**之後**

**設計：** 淺色底色小橫條，放在卡片底部

格式範例：
```
📄 客戶 PO (PDF/Excel)  →  出貨單 Excel（含稅額）✅
```

各卡對照表：

| 技能 | 輸入 | 輸出 |
|---|---|---|
| `/名片` | 📷 名片照片 | Excel 通訊錄（自動去重） |
| `/追蹤信` | 📝 會議記錄 | 每人專屬提醒 Email（Word） |
| `/主管摘要` | 📂 多月報表 Excel | 一頁 A4 主管摘要（Word） |
| `/報帳` | 📷 發票照片 | 年度報帳 Excel（流水號延續） |
| `/白板` | 📸 白板照片 | 結構化 Excel 時程甘特表 |
| `/交接` | 📂 工作資料夾 | 完整交接說明書（Word） |
| `/報價` | 📄 舊報價 + 新需求說明 | 新報價單 Excel（原格式） |
| `/LINE總結` | 💬 LINE 匯出 .txt | 會議總結 + 分工清單（Word） |
| `/出貨` | 📄 客戶 PO (PDF/Excel) | 出貨單 Excel（含稅額） |
| `/規格翻譯` | 📋 中文規格書 .docx | 英文技術文件 Word（含封面） |
| `/客訴` | 📝 客訴信 / LINE截圖 | 回覆信（Word）+ 追蹤表（Excel） |
| `/BOM比對` | 📊 舊版 + 新版 BOM Excel | 標色差異分析報告（Excel） |

**通用卡片背景色：** `#f0fdf4`（綠色系）
**製造業卡片背景色：** `#fefce8`（黃色系）

**設計：** 三個並排卡片，各有圓形數字標籤

### 章節一｜00:00 引言

影片開場點出核心痛點：

- ChatGPT 等傳統 AI 工具只能「回答問題」，每次都要從頭解釋
- Claude Cowork + Skills 的組合才是真正的「**AI 自動化工作系統**」
- 目標：打造一套可以處理 99% 重複工作的個人 AI 工作流

### 章節二｜01:43 什麼是 Claude Cowork

**核心概念：Cowork ≠ 一般聊天 AI**

| 傳統聊天 AI（ChatGPT 等） | Claude Cowork |
|---|---|
| 只能在對話框裡回答 | 能讀取、寫入本機資料夾 |
| 每次都要重新上下文 | 有記憶、有工作區 |
| 告訴你「怎麼做」 | 直接幫你「做完」 |
| 產出文字 | 產出真實的 Excel / Word / PDF |

**關鍵特性：**
- 可指定「工作資料夾」，AI 直接在裡面讀寫檔案
- 支援 Projects（專案隔離）、Scheduled（定時排程）、Dispatch（批次派送）
- 可安裝 Skills（技能）作為可呼叫的 SOP 模組

---

### 章節三｜02:13 背景設定（Custom Instructions）

**這是影片的第一個核心示範：** 給 AI 建立「角色設定」

**步驟：**
1. 進入 Claude Cowork → 點選 **Customize**
2. 填入「背景設定（Custom Instructions）」
3. 設定 AI 的角色、溝通風格、工作原則

**孔老師的示範設定內容包含：**
- AI 的身份定位（例：我的個人品牌助理）
- 優先使用的語言（繁體中文 / 簡體中文）
- 回應的語氣與格式要求
- 需要了解的用戶背景資訊

> **💡 對課程的應用啟示：**
> 這對應我們「技能工具庫」中「**Skills 是預裝工作記憶**」的概念。
> 差別在於：Custom Instructions 是全局設定；Skills 是任務級別的 SOP 召喚。

---

### 章節四｜04:25 自動化任務：自動生成晨間簡報

**這是影片最震撼的實際示範之一**

**任務描述：**
只輸入一句指令，Claude Cowork 自動完成以下多步驟工作：

```
指令：「幫我生成今天的晨間工作簡報」
```

**AI 自動執行的流程：**
1. 讀取工作資料夾裡的相關檔案（行程、待辦、上週報告）
2. 分析今日重點事項
3. 產出結構化晨間簡報（含：今日優先任務、注意事項、需跟進項目）
4. 自動儲存到指定資料夾

> **💡 對課程的應用啟示：**
> 這正是「**Skill = 教一次，用一輩子**」的最佳示範。
> 安裝好 `/晨報` 技能後，每天只需一個指令，所有步驟自動完成。

---

### 章節五｜08:06 Skills × Cowork（技能與工作區的結合）

**這是影片的核心重點，也是我們「技能工具庫」頁面的主題**

**Skills 的本質說明：**

> Skill 就是你預先寫好的「指令書（SOP）」，讓 AI 知道：
> - 這個任務的輸入是什麼格式？
> - 需要執行哪些步驟？
> - 輸出的格式和規則是什麼？

**孔老師的實際示範（Skills × Cowork 組合）：**

任務：**同時研究多個主題並整合報告**

```
步驟 1：安裝自訂 Skill（預先設定好研究流程 SOP）
步驟 2：在 Cowork 輸入 /research + 主題清單
步驟 3：Claude 自動：
         ├── 子任務 A：搜尋主題一的資料
         ├── 子任務 B：搜尋主題二的資料
         └── 主任務：整合所有子任務結果 → 輸出報告
```

**關鍵洞察：**
- Skill 讓 AI「知道規則」，Cowork 讓 AI「有能力執行」
- 兩者結合 = 完整的自動化工作流

---

### 章節六｜11:58 子代理功能（Sub-agent）

**影片中最「科幻感」的功能展示**

**示範場景：** 同時研究 10 個不同主題

- Claude 會自動「拆分任務」，建立多個子代理（Sub-agent）
- 10 個子代理**同時並行執行**，各自負責一個主題
- 完成後自動匯整成一份綜合報告

**實際輸出效果：**
- 10 個主題的研究在數分鐘內完成
- 每個主題有獨立的摘要
- 最終有整合性的比較分析

> **💡 對製造業的應用想像：**
> 同時比對 10 個供應商的報價 → 自動產出採購比較表
> 同時掃描多個月的銷售數據 → 自動產出異常警示報告

---

### 章節七｜13:52 互動圖表與流程圖

**技術亮點展示**

Claude Cowork 可以直接生成**可互動的 HTML 圖表**，不只是靜態圖片：

- **互動式圓餅圖**：點擊可展開細項
- **動態流程圖**：節點可拖曳、縮放
- **時序圖**：自動根據資料排列時間軸

**生成方式：**
```
指令：「幫我把這份 Excel 銷售數據做成互動式圓餅圖」
輸出：一個 .html 檔，打開瀏覽器就能直接互動操作
```

### 章節八｜16:02 總結

**孔老師的核心結論：**

1. **Claude Cowork 是「執行工具」，不只是「諮詢工具」**
   - 它能直接操作本機檔案，產出真實成果

2. **Skills 是 Cowork 的「超能力外掛」**
   - 每個 Skill 都是一個可反覆呼叫的自動化工作流

3. **三步建立個人 AI 工作系統：**
   - Step 1：設定 Custom Instructions（建立 AI 角色）
   - Step 2：安裝多個 Skills（預載你的工作 SOP）
   - Step 3：用 Cowork 執行日常任務（讓 AI 真正動手）

4. **最適合的使用場景：**
   - 有固定格式的重複文書作業
   - 需要整合多份資料的分析工作
   - 定期產出的報告類任務
